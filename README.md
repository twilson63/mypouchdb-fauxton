# Mysql PouchDB Fauxton

This is a simple way to access the MySQL PouchDB Database for a MySQL PouchDB Application.

Clone the repo

create a .env file

```sh
MYSQL_URI=mysql://user:password@domain:port/dbname
```

## Setup

```
yarn
```

## Start

```
yarn start
```

## Open

```
open http://localhost:6984/_utils
```
